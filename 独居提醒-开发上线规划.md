## 阶段 2：核心链路联调（第2-3周）

### 小后接口清单

| 接口 | 方法 | 说明 |
|------|------|------|
| `/auth/sms-code` | POST | 发送短信验证码 |
| `/auth/login` | POST | 手机号登录 |
| `/user/profile` | GET/PUT | 用户信息 CRUD |
| `/contacts` | GET/POST/PUT/DELETE | 紧急联系人管理 |
| `/contacts/:id/verify` | POST | 联系人手机号验证 |
| `/reminder/config` | GET/PUT | 提醒时间窗口配置 |
| `/reply/today` | POST | 今日回复 |
| `/reply/today` | DELETE | 撤回回复（30s内） |
| `/reply/status` | GET | 当前守护状态 |
| `/reply/streak` | GET | 连续平安天数 |

### 小前页面清单

| 页面 | 功能点 |
|------|--------|
| **引导流程** | 登录 → 协议 → 基本信息 → 联系人 → 提醒时间 → 授权 → 完成 |
| **首页** | 5种状态切换、大圆按钮、连续天数、倒计时 |
| **联系人管理** | 列表、添加表单、验证、删除、排序 |
| **提醒设置** | 时间窗口选择器 |

### 联调验证点

| 场景 | 验证点 |
|------|--------|
| 引导流程完整性 | 5步引导数据正确提交、状态正确流转 |
| 首页状态机 | idle/waiting/replied/grace/alert 正确切换 |
| 撤回回复 | 30s 倒计时、接口调用、状态回滚 |
| 联系人验证 | 验证码发送、输入、校验、状态更新 |

## 阶段 3：告警链路与增值功能（第4-5周）

### 小后功能与接口

| 功能 | 接口 |
|------|------|
| **超时检测引擎** | 服务端定时任务（窗口截止+30分钟检查） |
| **告警触发** | `POST /alert/trigger` + 状态机管理 |
| **SMS 短信** | 集成阿里云/腾讯云短信（≤0.08元/条） |
| **语音电话** | 付费用户语音通知（可选：Phase 1 仅短信） |
| **联系人确认** | `POST /alert/:id/confirm` + `POST /alert/:id/help` |
| **告警时间线** | `GET /alert/:id/timeline` |
| **关怀看板** | `GET /dashboard/:guardianId` + 7天日历数据 |
| **子女端** | `POST /guardian/create` + 邀请链接生成 |
| **代确认** | `POST /guardian/:id/proxy-confirm` |

### 小前页面清单

| 页面 | 功能点 |
|------|--------|
| **联系人通知页** | 时间线、倒计时、确认/帮助按钮 |
| **需要帮助页** | 一键拨号、120、地址、付费引导 |
| **确认安全页** | 处理记录、返回首页 |
| **关怀看板** | 平安日历、统计数据、代确认、付费版/免费版 |
| **子女端流程** | 为父母开通（3步：填写/分享/等待绑定） |

### 小后关键技术挑战

| 挑战 | 方案 |
|------|------|
| 超时检测可靠性 | 用 Redis 延迟队列或 AWS SQS + Lambda |
| SMS 成本控制 | 免费用户仅短信，付费用户短信+语音 |
| 告警状态一致性 | 状态机 + 数据库事务 |
| 实时性 | WebSocket 或 SSE 推送首页状态变更 |

## 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 后端 API 延迟 | 阻塞前端联调 | 用 Mock 数据先行开发，接口就绪后切换 |
| 短信服务审批慢 | 告警链路无法测试 | 先用邮件通知替代，上线前切换 |
| Apple IAP 审核 | 付费功能无法验证 | 先用测试支付，沙盒环境验证 |
| 通知权限拒绝率高 | 引导流程无效 | 增加二次确认 + 完成页状态区分 |
